Detailed Description

LIFO (stack) implementation of tree visiting algorithm. A TreeTraverse algorithm will operate on a tree structured rooted at TreeNodePtr with help of TreeNodeStack with respect to which traversal strategy

  1. In-order traversal
  2. Post-order traversal
  3. Pre-Order traversal
  4. Breadth-first traversal
  5. Depth-first traversal
